# DPの実際の利用

## 集積回路設計

集積回路（IC）の設計において、動的計画法（DP）は様々な最適化フェーズで重要な役割を果たしています。

### 集積回路におけるDPの利用例

1. **テクノロジマッピング（回路のゲート割り当て）**
   論理合成の最終段階であるテクノロジマッピングでは、回路の論理関数をセルライブラリ内の実際のゲート（NANDやインバータなど）に割り当てます。この際、回路を「サブジェクトグラフ」と呼ばれる2入力の木構造に分解し、それをセルの「パタングラフ」で覆う「Tree Covering（木の被覆問題）」として定式化されます。この被覆問題において、面積や遅延を最小化するためにDPが用いられます。
2. **バッファ挿入（配線遅延の最適化）**
   LSIの微細化に伴う配線遅延や、最大容量制約・セットアップ違反を解消するために、配線上にバッファを挿入します。代表的な「Van Ginnekenのアルゴリズム」やその派生手法では、配線を木構造と捉え、終端（シンク）から始点（ソース）へ向かうボトムアップのDPによって最適なバッファ位置とサイズを決定します。
3. **フロアプランと配置**
   長方形のモジュールを二次元平面上に重ならないように配置する問題において、順列対（Sequence-Pair）で表現された相対位置情報から、配置の幅や高さ（配置コスト）を最小化する実際の座標を決定する際にもDPが利用されます。

### 競技プログラミング風の問題文の考察

テクノロジマッピングにおける「Tree Covering」のアルゴリズムは、競技プログラミングにおいて **「木DP」**や、計算済みの領域から結果を受け取る **「貰うDP」**と呼ばれる典型的なDPの手法に非常によく似ています。これらを参考に、AtCoder風の問題文と解法を考察します。

### 問題
1. **【問題名】 Gate Covering on a Tree**

**【問題文】**
集積回路の設計では、抽象的な論理回路を、実際に工場で製造可能な「基本ゲート」の組み合わせに変換する必要があります。
今、入力から出力へ向かって信号が流れる、頂点数 $N$ の根付き木 $G$（サブジェクトグラフ）が与えられます。頂点 $1$ を根（最終出力）とし、葉を初期入力とします。
また、あなたが使用できる基本ゲートの設計図として、$M$ 種類の木構造をしたパタングラフ $P_1, P_2, \dots, P_M$ が与えられます。各パタングラフ $P_i$ を回路に組み込むと、面積コストとして $W_i$ がかかります。

木 $G$ のすべての頂点を、これらのパタングラフの集合を用いて隙間なく被覆（カバー）したいとします。ただし、あるパタングラフを割り当てたとき、そのパタングラフの「入力（葉）」となる頂点は、別のパタングラフの「出力（根）」と一致していなければなりません。
木 $G$ を被覆するために必要な面積コストの総和の最小値を求めてください。

**【制約】**
* $1 \le N \le 10^5$
* $1 \le M \le 100$
* 各パタングラフの頂点数は $10$ 以下
* $W_i$ は正の整数

---

2. **【問題名】 Timing-Driven Gate Covering**

**【問題文】**
集積回路を正常に動作させるためには、回路の面積を小さくするだけでなく、与えられた「目標クロック周期 $T$」に間に合うように信号が伝わる必要があります。

頂点数 $N$ の根付き木 $G$（サブジェクトグラフ）が与えられます。葉を初期入力、根（頂点 $1$）を最終出力とします。
また、使用できる $M$ 種類のパタングラフ $P_1, P_2, \dots, P_M$ が与えられます。各パタングラフ $P_i$ を回路に組み込むと、 **「面積コスト $W_i$」** がかかり、さらに信号が通過する際の **「遅延コスト $D_i$」** が発生します。

木 $G$ をパタングラフで隙間なく被覆したとき、ある入力（葉）から出力（根）までの経路における遅延コストの総和を「経路の遅延」と呼びます。回路全体の **「最大遅延」** を $D_{max}$ としたとき、 **「遅延余裕（Slack）」** は $T - D_{max}$ で計算されます。

遅延余裕が $0$ 以上（つまり $D_{max} \le T$）となるような被覆の中で、 **面積コストの総和が最小となるような最小面積** と、その被覆を達成したときの **最大遅延余裕** を出力してください。もし、どのように被覆しても遅延余裕が $0$ 以上にならない場合は `IMPOSSIBLE` と出力してください。

**【制約】**
* $1 \le N \le 10^4$
* $1 \le M \le 100$
* $1 \le T \le 1000$ （最大クロック周期）
* $W_i, D_i$ は正の整数
