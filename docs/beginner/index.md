# C++競プロ入門者用問題ページ

## はじめに

このページは[アルゴリズム解説のページ](https://20s.d3bu.net/algorithm/)の執筆に疲れたときの気分転換用にこそっと書き始めたページです．自作問題とそれらの答えを載せていくつもりです．

あと問題の解説ページではいろんなヘッダファイルを使っていますが`#include<bits/stdc++.h>`を書けばそんなことしなくて済むので皆さんはこっちを使いましょう．

## レベル0 C++始めたての人用問題

この章ではC++始めたての人に向けた問題を掲載していきます．

ちなみにC++を学ぶのであれば[APG4b](https://atcoder.jp/contests/APG4b)がおすすめです．

### 問題0-1

```
整数Aが与えられるので，「デカビタの年齢はA歳です」の形で年齢を出力してください．

制約
1<=A<=120
Aは整数である

入力
A
```

入力例1

```
18
```

出力例1

```
デカビタの年齢は18歳です
```

入力例2

```
1
```

出力例2

```
デカビタの年齢は1歳です
```

[解答例](anser/index.md#問題0-1)

## レベル1 ABC-A問題を解きたい人用問題

大体dif 0~20 ぐらいの問題を出すつもりです．

### 問題1-1

```cpp
文字列Sが与えられます．この文字列が"d3bu"であればYesを，そうでなければNoを出力してください．

制約
1<=|S|<=100
Sは英数字からなる文字列である

入力
S
```

入力例1

```
d3bu
```

出力例1

```
Yes
```

入力例2

```
kcct
```

出力例2

```
No
```

入力例3

```
d3Bu
```

```
No
```

大文字小文字も区別します．

### 問題1-3

```
神戸高専では成績が60点未満の教科が4つ以上か，30点未満の教科が1つ以上あれば留年してしまいます．
また平均点が60点未満でも留年です．
この学校に通うデカビタ君の国語の成績はA点，数学の成績はB点，物理の成績はC点，歴史の成績はD点，英語の成績はE点です．
これら5教科だけを見た時，デカビタ君が進級できるかどうかを出力してください．

制約
0<=A,B,C,D,E<=100
入力はすべて整数

入力
A B C D E

出力
進級できるのであればYes 出来なければNoを出力
```

入力例1

```
51 67 83 54 92
```

出力例1

```
Yes
```

入力例2

```
45 59 60 42 51
```

出力例2

```
No
```
60点未満が4つあるため進級できません．

入力例3

```
100 100 100 100 29
```

出力例3

```
No
```

30点未満が一つでもあれば留年です．

入力例4

```
60 47 60 60 51
```

出力例4

```
No
```

平均点が60点未満なので留年です．

[解答例](anser/index.md#問題1-3)

## レベル2 ABC-B問題を解きたい人用問題

この章ではB問題を解きたい人用問題を載せていきます．

問題2-1

```
長さNの整数列Aが与えられます．この数列の中央値を出力してください．

制約
1<=N<=2×10^5
-10^9<=A_i<=10^9
入力はすべて整数

入力
N
A_1 A_2 ... A_N
```

入力例1

```
9
6 8 4 4 1 2 5 9 5
```

出力例1

```
5
```

入力例2

```
6
7 4 6 2 5 9
```

出力例2

```
5.5
```

長さが偶数の時は真ん中に最も近い2つの要素の平均が中央値です．

[解答例](anser/index.md#問題2-1)
